# FCR 计算工具 — 用户说明书与质控标准

---

## 第一部分：软件说明

### 1.1 软件概述

本工具基于**浙江大学博士学位论文（2025）**《猪饲料转化效率和采食行为的 GWAS 及基因组选择研究》中提出的 FCR 计算方法开发，将论文中的数据清洗、体重质控、ADG 回归和 FCR 计算的完整流程固化为可交互的桌面应用程序。

### 1.2 适用范围

- **目标用户**：猪场生产管理人员、育种数据分析人员
- **输入数据**：电子饲喂站（FIRE 系统、9ZC-170 等）导出的原始 CSV 数据
- **输出结果**：
  - 每头猪的清洗后数据明细（Excel）
  - 全部猪的 FCR 汇总表（Excel）
  - 每头猪的体重-日龄拟合曲线图（PNG）

### 1.3 运行方式

#### 方式一：Python 源码运行
```bash
cd zheda_FCR
python fcr_main.py
```

#### 方式二：exe 可执行文件运行
双击 `FCR_Calculator.exe` 即可（无需安装 Python）。

软件接收的 CSV 文件必须包含以下 **9 列**：

| 列名 | 含义 | 类型 | 示例 |

## 第二部分：质控标准

### 2.1 质控流程总图

```
原始数据
    │
    ▼
[Step 1] 个体层面预过滤
    ├── 测试日龄 < 100 天 或 > 200 天 → 标记
    ├── 体重 < 25 kg 或 > 180 kg → 剔除单条记录
    └── 初测体重超出 25~45 kg 范围 → 标记异常样本
    │
    ▼
[Step 2] 采食数据错误过滤（Casey 2005 准则）
    ├── 10 种可计算错误类型（FIV / OTV / FRV 相关）
    └── 命中任一错误 → 记录标记，该条采食量不参与计算
    │
    ▼
[Step 3] 5 步体重质控（浙大博士论文核心方法）
    ├── ① 粗过滤：体重 > 180 kg 或 < 25 kg
    ├── ② ADG 参考值计算：起止体重差 / 天数
    ├── ③ 预测体重：start_wt + 天数 × ADG_ref
    ├── ④ 极端错误：|实际体重 − 预测体重| > ADG_ref × 21
    └── ⑤ 标准差过滤：当天 2 倍标准差 + 7 倍 ADG_ref
    │
    ▼
[Step 4] ADG 计算 → 仅使用通过质控的有效体重做线性回归
    │
    ▼
[Step 5] ADFI 计算 → 总有效采食量 / 测试天数
    │
    ▼
[Step 6] FCR 计算 → ADFI / ADG（料重比）
```

### 2.2 初测体重质控标准

| 参数 | 默认值 | 可调范围 | 说明 |
|:-----|:------:|:--------:|:------|
| 初测体重下限（init_lower） | 25 kg | 10~60 kg | 首个有效体重小于此值 → 标记异常 |
| 初测体重上限（init_upper） | 45 kg | 20~100 kg | 首个有效体重大于此值 → 标记异常 |
| 初测体重严重偏离阈值 | init_upper + 10 kg | 自动计算 | 首个体重 > 上限+10 → 以第 2 次测定体重为起始参考 |

**处理逻辑**：
1. 若 `初测体重 < 25` 或 `初测体重 > 45` → 在该条记录的 `bw_remark` 中注明，并标记 `is_abnormal = True`（该记录不参与 ADG 回归）
2. 若 `初测体重 > 45 + 10 = 55 kg` → 在 ADG 参考值计算中，将起始体重自动回退到第 2 次测定体重值，避免单个异常值污染参考线
3. 在汇总表中将该猪只标记为 `异常样本`，`fill_info` 中注明原因

### 2.3 采食数据错误类型与阈值

基于 Casey, Stern, Dekkers (2005) 提出的 16 种错误类型，本软件实现了其中基于可用数据列的 **10 种**：

| 错误类型 | 变量 | 条件 | 阈值 | 原因 |
|:--------:|:----:|:-----|:----:|:-----|
| FIV-lo | 次采食量 | 所有访问 | < -20 g | 负采食量（出料槽重 > 入料槽重） |
| FIV-hi | 次采食量 | 所有访问 | > 2,000 g | 单次采食量过高 |
| FIV-0 | 次采食量 | OTV = 0 s 的访问 | \|FIV\| > 20 g | 零时长但有采食量 |
| OTV-lo | 次采食时长 | 所有访问 | < 0 s | 退出时间早于进入时间 |
| OTV-hi | 次采食时长 | 所有访问 | > 3,600 s | 采食时间超过 1 小时 |
| FRV-hi-FIV-lo | 采食速率 | 0 < FIV < 50 g | > 500 g/min | 小采食量但速率极高 |
| FRV-hi-strict | 采食速率 | FIV ≥ 50 g，前后有负采食量 | > 110 g/min | 邻近异常时的严格判断 |
| FRV-hi | 采食速率 | FIV ≥ 50 g，前后正常 | > 170 g/min | 常规速率上限 |
| FRV-0 | 采食速率 | FRV = 0 g/min | OTV > 500 s | 零速率但占用时长过长 |
| FRV-lo | 采食速率 | FRV ≠ 0 g/min | ≤ 2 g/min | 采食速率过低 |

> **说明**：原始 Casey 2005 论文中还包含 LWD-lo/hi、FWD-lo/hi、LTD-lo、FTD-lo 等 6 种错误类型，这些需要**料槽称重传感器**的入口/出口重量数据。本输入 CSV 仅有每次访问的`采食量`和`猪体重`，缺少料槽重量数据，因此无法计算这 6 种类型。

### 2.4 5 步体重质控标准

#### 第 1 步：粗过滤
| 条件 | 处理 |
|:----|:------|
| 体重 > 180 kg | 该记录标记为异常 |
| 体重 < 25 kg | 该记录标记为异常 |

#### 第 2 步：ADG 参考值
```
ADG_ref = (结测体重 − 开测体重) / (结测日龄 − 开测日龄)
```
- 开测/结测体重使用**站外校正秤**读数（砝码每日校准，读数稳定后记录）
- 若初测体重 > 上限 + 10 kg，自动以第 2 次测定体重代替起始体重

#### 第 3 步：预测体重
```
predicted_weight = start_weight + (test_age − start_age) × ADG_ref
```
- 对每条访问，根据时间推算当天的预期体重

#### 第 4 步：极端错误过滤
```
|实际体重 − 预测体重| > max(ADG_ref × 21, 10)
```
- 超过此范围的记录标记为 `极端错误`
- 21 天为合理波动的最大天数窗口（论文中使用）

#### 第 5 步：标准差二次过滤
```
(a) |体重 − 当天均值| > 2 × 当天标准差
(b) |体重 − 当天均值| > ADG_ref × 7
```
- 命中任一条件 → 标记为 `回归偏差较大`
- 当天的记录数 < 3 条时跳过此步

### 2.5 ADG 计算标准

**方法**：全部通过 5 步质控的有效体重对日龄做**线性回归**

```
BW = β₀ + β₁ × age + ε
ADG = β₁
```

**回归要求**：
- 至少 3 条有效体重记录
- 日龄有足够变异（标准差 > 0）
- 回归结果同时给出 R² 作为拟合优度指标

### 2.6 FCR 计算标准

```
FCR（料重比）= ADFI / ADG
```
即 **Feed:Gain**（饲料消耗量 / 体重增量），单位 kg/kg。

保留 **3 位小数**。无效值（None 或 ≤ 0）标记为异常样本。

### 2.7 异常样本判定标准

满足以下任一条件的猪只，在汇总表中标记为 `异常样本`：

| 条件 | 判定依据 |
|:-----|:---------|
| 初测体重超出 25~45 kg 范围 | `bw_remark` 中含 "初测体重" + "需在" 关键字 |
| 数据不足 | 质控后有效记录 < 5 条 |
| FCR 无效 | FCR 为 None 或 ≤ 0 |
| ADG 回归失败 | 有效体重 < 3 条或日龄无变异 |

异常样本的 FCR 值仍会计算（若可能），但在汇总表中明确标注 `异常样本`，用户可根据实际需要决定是否纳入分析。

### 2.8 FCR  参考范围

| 品种 | 均值（kg/kg） | 标准差 | 遗传力 h²(SE) |
|:-----|:------------:|:------:|:-------------:|
| 杜洛克（DD） | 2.42 | 0.33 | 0.24 (0.03) |
| 长白（LL） | 2.41 | 0.31 | 0.36 (0.04) |
| 大白（YY） | 2.49 | 0.34 | 0.29 (0.02) |
