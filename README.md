# FCR 批量计算工具 v1.2 — UI 美化方案

## 概述

对 FCR 批量计算工具的 GUI 层（`fcr_app/gui.py`）进行了全面的 UI 设计系统重构，采用 COFCO 品牌绿色配色方案，构建了一套基于 QSS 的专业桌面界面设计系统。

## 关键变更

### 1. 全局 QSS 设计系统（新增 `APP_STYLESHEET`）
- **配色方案**：COFCO 品牌绿 `#1B7340` 为主色，中等饱和度，覆盖 Primary / Hover / Light / Surface / Danger / Log-bg / Card 七个语义色阶
- **间距体系**：8pt 网格（4 / 8 / 12 / 16 / 24 / 32 px），统一应用于布局边距和控件间距
- **字体层级**：17px 标题 → 13px 正文 → 12px 信息 → 11px 提示/日志，两种字重（400 / 500）
- **圆角体系**：4px（小控件）→ 6px（按钮/输入框）→ 8px（卡片/面板）→ 10px（徽章）

### 2. 布局结构重构（`MainWindow._init_ui()`）
- **品牌头栏**：白底 + 绿色下边框，左侧应用标题与英文副标题，右侧绿色版本徽章
- **卡片式分区**：数据输入区、计算参数区各自独立为白底圆角卡片，带绿色标题头和分隔线
- **预览区改进**：左右面板各包裹在 section_card 中，标题头统一风格
- **日志面板**：深绿黑色背景 `#1B2E23`，等宽字体，彩色日志级别区分
- **状态栏**：白底 + 顶部细线分隔，进度条固定宽度

### 3. 组件样式增强
| 组件 | 改进内容 |
|------|---------|
| **按钮** | 三种变体：primary_btn（绿底白字）、danger_btn（红底白字）、默认（白底灰边） |
| **输入框** | 焦点态绿色边框 + 轻微背景变化，placeholder 灰色提示 |
| **SpinBox** | 自定义上下箭头按钮（绿色三角），hover 绿底 |
| **复选框** | 18px 圆角方框，选中态绿色填充 |
| **进度条** | 绿色填充 + 浅灰轨道，圆角设计 |
| **树形列表** | 交替行色，选中绿色高亮白字，hover 浅绿背景 |
| **滚动条** | 8px 细条，圆角，hover 变绿 |
| **分割器** | 3px 细线，hover 变绿 |

### 4. 日志处理器增强（`QLogHandler`）
- 使用 `appendHtml` 替代 `appendPlainText`
- 按日志级别着色：INFO 浅绿、WARNING 琥珀、ERROR 浅红、CRITICAL 亮红
- HTML 特殊字符自动转义，防止注入

### 5. 图片预览组件改进（`ImageLabel`）
- 使用 QSS objectName `image_label` 统一样式
- 圆角边框 + 浅色背景
- 改进的占位符文案

## 保持不变的部分
- **Worker 线程类**：完全保持原逻辑不变
- **所有方法签名**：MainWindow 的 18 个方法签名完全一致
- **所有信号/槽连接**：信号定义和连接逻辑完全保留
- **core.py / chart.py / __init__.py**：未做任何修改
- **fcr_main.py**：未做任何修改
